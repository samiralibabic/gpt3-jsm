# AI landing page

![screenshot](screenshot.png)

A landing page for a GPT powered Webapp, built with React.

## Clone repository

```shell
git clone https://github.com/samiralibabic/gpt3-jsm.git
cd gpt3-jsm
```

## Install dependencies

```shell
npm install
```

## Run

```shell
npm start
```

Open `http://localhost:3000/gpt3-jsm`
